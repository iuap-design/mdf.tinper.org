### **功能说明**
文本输入框，可通过扩展属性配置成多行文本框<br />**
<a name="J0shL"></a>
### **属性列表**
_无_<br />**
<a name="zD1eV"></a>
### 扩展属性 cStyle
| **属性** | **类型** | **默认值** | **含义** | **备注** |
| :---: | :---: | :---: | :---: | :---: |
| type | Text | null | 输入类型 | 值为 textarea 时是多行文本，如：{"type":"textarea"} |
| tips | Text | null | 提示内容 | Tips提示内容，如：{"tips":"ID必输"} |
